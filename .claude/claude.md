# Advancing Merch App - Claude Context

## Project Overview

This is a music act management application handling two primary functions:
1. **Merch Management** - Inventory and sales tracking
2. **Advancing** - Tour planning and AI-assisted documentation

## Documentation

For detailed architecture, database schema, and integration plans, see:
- [Architecture Documentation](./.claude/architecture.md)

## Key Technologies

- **Frontend:** React, TypeScript, Tailwind CSS, Vite
- **Backend:** Lovable Cloud (Supabase)
- **Database:** PostgreSQL
- **AI:** Lovable AI (Google Gemini models)

## Development Notes

- Minimal UI scaffold currently implemented
- Focus on data structure and schema first
- UI development will follow after data flows are established
- All backend changes deploy automatically
- Frontend changes require clicking "Update" in publish dialog
